# <a name="review-the-second-section"></a>复习第二部分
让我们复习已了解的内容。

使用 **Microsoft Flow** 可以轻松完成以下操作：

* 在移动设备上获取**推送通知**以响应有关你喜爱的 Web 服务的事件。
* 在联机存储服务之间**复制和移动文件**。
* 从服务**收集数据**以及遥测信息。
* **使用 SharePoint 创建审批工作流**。
